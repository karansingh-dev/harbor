# Harbor

Turn a Linux VPS into a simple application platform.

Harbor is a self-hosted deployment tool that helps developers deploy and manage applications on their own servers without manually configuring infrastructure.

## Vision

Deploying an application to a VPS should be simple.

Instead of manually setting up Docker containers, managing processes, tracking deployments, and remembering server commands, Harbor provides a guided workflow that helps developers get applications running quickly.

## Status

🚧 Early development

Harbor is currently in active development and not ready for production use.
